3-Dars: O'zgaruvchilar va Ma'lumot Turlari

O'zgaruvchilar

O'zgaruvchilar dasturda ma'lumotlarni saqlash uchun ishlatiladi. Python tilida o'zgaruvchini yaratish juda oddiy: 

x = 5
y = "Salom, Dunyo!"

Yuqoridagi kodda x o'zgaruvchisiga 5 qiymati berilgan, y o'zgaruvchisiga esa matnli ma'lumot saqlanadi.

Ma'lumot Turlari

Python tilida ko'plab ma'lumot turlari mavjud. Eng ko'p ishlatiladigan ma'lumot turlari:

  • int — butun sonlar (masalan, 5)
  • float — o'nlik sonlar (masalan, 3.14)
  • str — matnli ma'lumotlar (masalan, "Salom")
  • bool — mantiqiy qiymatlar (True yoki False)

Misol kod

Quyida o'zgaruvchilar va ma'lumot turlaridan foydalanish bo'yicha oddiy misol keltirilgan: 

x = 10
y = 3.14
z = "Python"
is_active = True

print(x, y, z, is_active)

Kodni ishga tushirgandan so'ng, quyidagi natijani ko'rasiz:

10 3.14 Python True

Keyingi darslar

Keyingi darslarda shart operatorlari va sikllar bilan tanishasiz.
