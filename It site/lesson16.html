16-Dars: Tabiiy Tilni Qayta Ishlash (NLP)

Tabiiy Tilni Qayta Ishlashga kirish

Tabiiy Tilni Qayta Ishlash (NLP) inson tilini kompyuter orqali tahlil qilish va tushunishga imkon beradi. Ushbu darsda NLP'ning asosiy tushunchalarini o'rganamiz va sodda dasturlar yordamida uni qanday qo'llashni ko'rib chiqamiz.

Kerakli kutubxonalar

Python'da NLP bilan ishlash uchun quyidagi kutubxonalarni o'rnating: 

pip install nltk
pip install spacy
pip install textblob
  • NLTK (Natural Language Toolkit): Eng mashhur NLP kutubxonasi.
  • spaCy: Tezkor va kuchli NLP kutubxonasi.
  • TextBlob: Oddiy NLP operatsiyalari uchun foydalaniladigan qulay kutubxona.

Oddiy NLP Amaliyoti

Quyida NLTK yordamida oddiy matn tahlilini amalga oshiramiz: 

import nltk
from nltk.tokenize import word_tokenize
nltk.download('punkt')

text = "Salom, Python dasturchilari! NLP bilan ishlash qiziqarli."
words = word_tokenize(text)

print(words)

TextBlob Bilan Matn Tahlili

Quyida TextBlob yordamida matnni sentiment tahlilini amalga oshiramiz: 

from textblob import TextBlob

text = "Python juda qiziqarli dasturlash tili."
blob = TextBlob(text)

# Sentiment tahlili
sentiment = blob.sentiment
print(sentiment)

spaCy Bilan Matnni Lemmatizatsiya Qilish

spaCy yordamida matnni lemmatizatsiya qilish amaliyoti: 

import spacy
nlp = spacy.load("en_core_web_sm")

text = "We are learning natural language processing."
doc = nlp(text)

# Lemmatizatsiya
for token in doc:
    print(f"{token.text} -> {token.lemma_}")
