12-Dars: Avtomatlashtirish (pytest)

Avtomatlashtirish nima?

Avtomatlashtirish, dasturlar va jarayonlarni test qilishni avtomatik tarzda bajarish jarayonidir. Bu, dasturiy ta'minotni yanada sifatli qilish uchun muhimdir.

pytest kutubxonasi

pytest — Python dasturlarini test qilish uchun keng tarqalgan kutubxona. Bu kutubxona sintaksis va xususiyatlari bilan juda qulay.

Asosiy xususiyatlar

- Oson o'rnatish va ishlatish.
- Testlarni modul va sinf sifatida yozish.
- Turli darajadagi testlarni yozish (unit testlar, funksional testlar va boshqalar).

pytest ni o'rnatish

Python'da kutubxonalardan foydalanish uchun avval ularni o'rnatish kerak bo'ladi. `pytest` kutubxonasini o'rnatish uchun terminal yoki buyruq satrida quyidagi buyruqni kiriting: 

pip install pytest

Agar sizda pip o'rnatilmagan bo'lsa, siz Python bilan birga kelgan get-pip.py skriptini ishlatib o'rnatishingiz mumkin.

Terminalga kirish

Windows foydalanuvchilari uchun

Windows'da terminalga kirish uchun "Command Prompt" yoki "Powershell"dan foydalanishingiz mumkin. Quyidagi usullar orqali terminalga kirishingiz mumkin:

  1. Start tugmasini bosing va qidiruv maydoniga "cmd" yoki "Command Prompt" deb yozing. Keyin esa topilgan dastur ustiga bosib oching.
  2. Alternativ ravishda, klaviaturada Windows + R tugmalarini bosib, ochilgan oynaga "cmd" deb yozing va "Enter" bosing.

MacOS va Linux foydalanuvchilari uchun

MacOS va Linux tizimlarida terminalga kirish juda oson:

  1. MacOS: "Applications" bo'limiga kiring, "Utilities" papkasidan "Terminal"ni oching.
  2. Linux: Asosan "Ctrl + Alt + T" tugmalari orqali terminalni tezda ochishingiz mumkin.

pytest dan qanday foydalanish kerak?

Quyida pytest yordamida oddiy test yozish misoli keltirilgan: 

# test_foydalanuvchi.py

def test_qoshish():
    assert 1 + 1 == 2

def test_ayirish():
    assert 3 - 1 == 2

def test_multiply():
    assert 2 * 2 == 4

Testlarni ishga tushirish

Testlarni ishga tushirish uchun terminalda quyidagi buyruqni bajarish kerak: 

pytest test_foydalanuvchi.py
