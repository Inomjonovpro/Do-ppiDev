18-Dars: PIP va Python Kutubxonalari bilan ishlash

PIP haqida

PIP - Python uchun paket menejeri bo'lib, u orqali Python kutubxonalarini o'rnatish va boshqarish mumkin. Siz Python dasturlash tilida foydalanadigan kutubxonalarni PIP orqali tez va oson o'rnatishingiz mumkin.

PIP yordamida kutubxonalarni o'rnatish

PIP yordamida kutubxonalarni o'rnatish uchun terminal yoki buyruq qatoriga quyidagi ko'rinishda buyruq kiritiladi: 

pip install kutubxona_nom

Masalan, numpy kutubxonasini o'rnatish uchun: 

pip install numpy

PIP yordamida o'rnatilgan kutubxonalarni ko'rish

Kompyuteringizda o'rnatilgan Python kutubxonalarini ko'rish uchun quyidagi buyruqdan foydalanishingiz mumkin: 

pip list

Misol: NumPy bilan oddiy amallar

NumPy - bu matematik va statistik hisob-kitoblarni amalga oshirish uchun keng qo'llaniladigan Python kutubxonasi. Quyidagi misolda NumPy kutubxonasi yordamida oddiy arifmetik amallar bajariladi: 

import numpy as np

# NumPy yordamida massiv yaratish
massiv = np.array([1, 2, 3, 4, 5])

# Massiv elementlarini qo'shish
print(massiv + 5)

# Massivdagi elementlarning o'rtacha qiymatini hisoblash
print(np.mean(massiv))

Yana qanday kutubxonalar bilan ishlash mumkin?

Quyida Python'da keng qo'llaniladigan kutubxonalar ro'yxati keltirilgan:

  • pandas: Ma'lumotlarni tahlil qilish va ishlov berish uchun mo'ljallangan.
  • matplotlib: Grafik va diagrammalar yaratish uchun ishlatiladi.
  • scikit-learn: Mashina o'rganish modellari yaratish uchun ishlatiladi.
  • requests: API va internet sahifalaridan ma'lumotlarni olish uchun.
  • flask va django: Veb-ilovalar yaratish uchun.

Keyingi qadamlar

Endi PIP yordamida kutubxonalarni qanday o'rnatish va ularni ishlatishni o'rgandingiz. Keyingi darslarda Python yordamida ko'proq kutubxona va ularning imkoniyatlarini o'rganishni davom ettiramiz.
