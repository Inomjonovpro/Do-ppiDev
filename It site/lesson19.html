19-Dars: Mashhur Python Kutubxonalari bilan ishlash

1. NumPy bilan ishlash

NumPy Python'da matematik va ilmiy hisob-kitoblarni amalga oshirish uchun keng qo'llaniladi. U katta hajmdagi massivlar (arrays) bilan ishlashni juda osonlashtiradi. 

import numpy as np

# NumPy yordamida massiv yaratish
massiv = np.array([1, 2, 3, 4, 5])

# Massiv elementlarini ko'paytirish
massiv_kopaytirish = massiv * 2
print("Massivni ko'paytirish:", massiv_kopaytirish)

# Massivdagi elementlarning o'rtacha qiymatini hisoblash
ortacha_qiymat = np.mean(massiv)
print("O'rtacha qiymat:", ortacha_qiymat)

2. Pandas bilan ishlash

Pandas - ma'lumotlarni tahlil qilish va ular bilan ishlash uchun mo'ljallangan kutubxona. U ma'lumotlar ramkalari (DataFrames) bilan ishlash imkoniyatini beradi. 

import pandas as pd

# Pandas ma'lumotlar jadvali yaratish
data = {'Ism': ['Ali', 'Vali', 'G'ani'],
        'Yosh': [25, 30, 22]}

df = pd.DataFrame(data)

# Jadvalni ko'rsatish
print(df)

# Ma'lumotlar jadvalidagi eng katta yoshni topish
eng_katta_yosh = df['Yosh'].max()
print("Eng katta yosh:", eng_katta_yosh)

3. Matplotlib bilan diagrammalar yaratish

Matplotlib - diagramma va grafiklar chizish uchun juda foydali kutubxona. Uni ma'lumotlarni ko'rsatuvchi vizualizatsiyalar yaratish uchun ishlatish mumkin. 

import matplotlib.pyplot as plt

# Ma'lumotlar
yoshlar = [25, 30, 22]
ism = ['Ali', 'Vali', 'G\'ani']

# Diagramma chizish
plt.bar(ism, yoshlar)
plt.title('Yoshlar taqqoslanishi')
plt.xlabel('Ism')
plt.ylabel('Yosh')
plt.show()

PIP orqali kutubxonalarni o'rnatish

Yuqoridagi kutubxonalarni ishlatish uchun siz quyidagicha ularni PIP yordamida o'rnatishingiz mumkin:

  • pip install numpy - NumPy kutubxonasini o'rnatish.
  • pip install pandas - Pandas kutubxonasini o'rnatish.
  • pip install matplotlib - Matplotlib kutubxonasini o'rnatish.
