9-Dars: Funktsiyalar

Funktsiya nima?

Funktsiyalar — bu biror amallarni bajarish uchun yozilgan kod bo'laklaridir. Ular kodni qayta ishlatish imkonini beradi va dasturiy ta'minotni boshqarishni osonlashtiradi.

Funktsiyani qanday yaratish?

Funktsiyani yaratish uchun def kalit so'zidan foydalanamiz. Quyida oddiy funktsiya misoli keltirilgan: 

def salom_ber():
    print("Salom, Dunyo!")

Ushbu kodda salom_ber nomli funktsiya yaratilgan va u chaqirilganda "Salom, Dunyo!" matnini chiqaradi.

Funktsiyani chaqirish

Funktsiyani chaqirish uchun uning nomini yozib, qavs ochish va yopishimiz kerak:

salom_ber()

Parametrlar va argumentlar

Funktsiyalarga parametrlar qo'shish mumkin. Quyida parametrli funktsiya misoli keltirilgan: 

def salom_ber(isim):
    print(f"Salom, {isim}!")

Ushbu funktsiyani chaqirganda nom berishimiz kerak:

salom_ber("Muhammadamin")

Natija:

Salom, Muhammadamin!

Funktsiyalardan qaytarish qiymati

Funktsiyalardan qiymat qaytarish uchun return kalit so'zidan foydalanamiz. Quyida misol keltirilgan: 

def qo'sh(x, y):
    return x + y

Bu funktsiyani chaqirish orqali natijani olishimiz mumkin:

natija = qo'sh(5, 10)
print(natija)

Natija:

15
