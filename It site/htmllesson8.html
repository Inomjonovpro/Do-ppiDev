Responsive Dizayn Asoslari

Responsive Dizayn Nima?

Responsive dizayn – bu veb-sahifalarning turli ekran o'lchamlari va qurilmalarda yaxshi ko'rinishini ta'minlash uchun ishlatiladigan usuldir. Bu, saytni mobil telefonlardan tortib, planshetlar va kompyuterlarga moslashtirish imkonini beradi.

Media Queries

Media queries yordamida sahifa dizaynini turli o'lchamlar uchun sozlash mumkin. Masalan:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

Flexbox va Grid

Flexbox va CSS Grid yordamida responsive dizayn yaratish juda oson. Ular elementlarni ekran o'lchamiga qarab joylashtirish imkonini beradi.

Responsive Rasmlar

Responsive rasmlar uchun srcset atributidan foydalanish mumkin. Bu, brauzerga qaysi rasmni yuklashini aniqlashga yordam beradi:

<img src="small.jpg" srcset="medium.jpg 640w, large.jpg 1280w" alt="Responsive rasm">

Amaliy Misol

Responsive dizaynni yaratish uchun oddiy bir misol:

<div style="display: flex; flex-wrap: wrap;">
    <div style="flex: 1 1 300px;">1</div>
    <div style="flex: 1 1 300px;">2</div>
    <div style="flex: 1 1 300px;">3</div>
</div>
