HTML 1-Dars: Kirish va O'rnatish

HTML nima?

HTML (HyperText Markup Language) — bu veb-sahifalarni tuzish uchun ishlatiladigan til. U orqali veb-sahifa tarkibini belgilaymiz va brauzerga bu sahifaning qanday ko'rinishda bo'lishini ko'rsatamiz.

HTML dasturini o'rnatish

HTML yozish uchun maxsus dastur o'rnatishga ehtiyoj yo'q, chunki siz oddiy matn muharriridan foydalanib HTML yozishingiz mumkin. Masalan, Notepad yoki Visual Studio Code kabi muharrirlarni ishlatishingiz mumkin.

Birinchi HTML faylingiz

Quyida oddiy HTML hujjatining namunasi keltirilgan: 

<!DOCTYPE html>
<html lang="uz">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Birinchi sahifa</title>
</head>
<body>
    <h1>Salom, Dunyo!</h1>
</body>
</html>

Yuqoridagi kodni saqlab, brauzerda ochganingizda, "Salom, Dunyo!" matni ko'rsatiladi.
