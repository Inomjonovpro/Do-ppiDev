1-Dars: Python va Visual Studio Code o'rnatish

Pythonni o'rnatish

Python dasturlash tilini o'rnatish juda oson. Pythonning rasmiy veb-sayti bu yerda mavjud. O'z operatsion tizimingizga mos versiyasini yuklab olib, o'rnating.

Python o'rnatish jarayonining ko'rinishi.

Visual Studio Code (VS Code) o'rnatish

Visual Studio Code — bu Python va boshqa dasturlash tillarida dasturlash uchun juda qulay muharrir. Uni bu yerda yuklab olishingiz mumkin. Yuklab olgach, o'rnatish uchun quyidagi ko'rsatmalarga amal qiling:

  1. Yuklab olish tugagandan so'ng, yuklangan faylni oching va o'rnatishni boshlang.
  2. O'rnatish paytida "Add to PATH" opsiyasini tanlashni unutmang. Bu Python bilan ishlashni osonlashtiradi.
  3. O'rnatish tugagandan so'ng, Visual Studio Code-ni ishga tushiring.

Visual Studio Code o'rnatish jarayoni.

Python kengaytmasini o'rnatish

Visual Studio Code'ga Python bilan ishlash uchun kerakli kengaytmani o'rnatishingiz lozim. Buning uchun quyidagi qadamlarni bajaring:

  1. Visual Studio Code'da chap tomondagi plaginlar belgisi (Extensions) ni bosing.
  2. Qidiruv qutisiga "Python" yozing va Microsoft tomonidan ishlab chiqilgan Python kengaytmasini o'rnating.
  3. O'rnatish tugagandan so'ng, dasturingiz Python bilan ishlashga tayyor.

Visual Studio Code'da Python kengaytmasini o'rnatish.

Python bilan dastur yozish

Visual Studio Code o'rnatilgan va Python kengaytmasi qo'shilgan bo'lsa, dasturingizni yozishga tayyorsiz. Quyidagi kodni VS Code'da yozib ko'ring: 

print("Salom, Dunyo!")

Kodni saqlang va .py kengaytmasi bilan nomlang (masalan, hello.py). Keyin terminalda python hello.py komandasini ishga tushiring.

VS Code'da Python dasturining chiqishi: "Salom, Dunyo!"

Keyingi darslar

Keyingi darslarda o'zgaruvchilar, ma'lumot turlari va dasturlash printsiplari bilan tanishasiz.
