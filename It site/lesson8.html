8-Dars: OOP (Ob'ektga yo'naltirilgan dasturlash)

Ob'ektlar

OOP (Ob'ektga yo'naltirilgan dasturlash) dasturlash uslubi bo'lib, unda dastur ob'ektlar orqali tashkil etiladi. Har bir ob'ekt o'z xususiyatlari (data) va metodlari (funksiyalar) bilan birga keladi.

Sinf (Class)

Sinf - bu ob'ektlar yaratish uchun shablon. Sinf ichida biz ob'ektlarning xususiyatlarini va metodlarini aniqlaymiz. Masalan: 

class Avtomobil:
    def __init__(self, model, rang):
        self.model = model
        self.rang = rang

    def ma'lumotlar(self):
        return f"{self.model} model, {self.rang} rang."

Ob'ektlarni yaratish

Yuqoridagi sinfdan ob'ekt yaratish uchun, biz quyidagicha sintaksisdan foydalanamiz: 

my_car = Avtomobil("BMW", "qora")
print(my_car.ma'lumotlar())

Merosi

OOP-da meros olish orqali biz bir sinfdan boshqa sinfga xususiyatlarni o'zlashtirishimiz mumkin. Bu kodni ko'ring: 

class SportAvtomobil(Avtomobil):
    def __init__(self, model, rang, tezlik):
        super().__init__(model, rang)
        self.tezlik = tezlik

    def ma'lumotlar(self):
        return f"{self.model} model, {self.rang} rang, tezligi: {self.tezlik} km/soat."

Polimorfizm

Polimorfizm - bu bir xil metod nomi turli xil ob'ektlar tomonidan turli xil xulq-atvorini belgilash imkonini beradi. Masalan: 

class YengilAvtomobil(Avtomobil):
    def ma'lumotlar(self):
        return f"{self.model} yengil avtomobil."

class Og'irAvtomobil(Avtomobil):
    def ma'lumotlar(self):
        return f"{self.model} og'ir avtomobil."

Bu yerda `YengilAvtomobil` va `Og'irAvtomobil` sinflari o'zlarining `ma'lumotlar` metodlarini o'ziga xos tarzda belgilaydi.
