13-Dars: Web Dasturlash (HTML, CSS, JavaScript)

Web Dasturlash nima?

Web dasturlash — bu internetda ishlaydigan dasturlar yaratish jarayoni. Bu jarayon uch asosiy texnologiya bilan bog'liq: HTML, CSS, va JavaScript. Ushbu texnologiyalar birgalikda veb-sahifalarni yaratadi.

HTML: Asosiy tuzilma

HTML (HyperText Markup Language) veb-sahifalarning asosiy tuzilmasini yaratish uchun ishlatiladi. Quyidagi kod misoli veb-sahifaning asosiy HTML tuzilishini ko'rsatadi: 

<!DOCTYPE html>
<html>
    <head>
        <title>Mening Birinchi Veb Sahifam</title>
    </head>
    <body>
        <h1>Xush kelibsiz!</h1>
        <p>Bu mening birinchi veb sahifam.</p>
    </body>
</html>

CSS: Stil berish

CSS (Cascading Style Sheets) veb-sahifalarga dizayn va stil qo'shish uchun ishlatiladi. Misol uchun, biz sahifadagi matn rangini va shriftni o'zgartirishimiz mumkin: 

body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

h1 {
    color: #ff5722;
}

p {
    font-size: 18px;
}

JavaScript: Interaktivlik qo'shish

JavaScript yordamida sahifalarga interaktivlik qo'shishingiz mumkin. Masalan, tugmani bosganingizda sahifadagi matn o'zgaradi: 

<button onclick="matnniOzgartir()">Bos!</button>
<p id="matn">Bu matn o'zgaradi.</p>

<script>
function matnniOzgartir() {
    document.getElementById("matn").innerHTML = "Matn o'zgartirildi!";
}
</script>

HTML, CSS, va JavaScript birgalikda

Web dasturlashda HTML, CSS va JavaScript birgalikda ishlaydi. HTML veb-sahifani tuzadi, CSS uni stil qiladi, JavaScript esa sahifani interaktiv qiladi.

Do'ppiDev sayti ham HTML, CSS va JavaScript yordamida yaratilgan

Kutubxonalarni yuklab olish

Web dasturlashda ko'pincha turli JavaScript kutubxonalaridan foydalanamiz. Masalan, jQuery yoki React kabi kutubxonalarni ishlatishdan avval ularni yuklab olish kerak. Buning uchun, terminalda quyidagi buyruqlarni yozish mumkin: 

// jQuery ni yuklash uchun:
npm install jquery

// React ni yuklash uchun:
npm install react

Web dasturlarini yuklab olish uchun npm (Node Package Manager) dan foydalanishingiz mumkin. Agar npm sizda o'rnatilmagan bo'lsa, Node.js ni o'rnatishingiz kerak bo'ladi.

Terminalda ishlash

Web dasturlash jarayonida ko'pincha terminal orqali buyruqlar bajariladi. Agar sizda terminal ochilmagan bo'lsa, uni yuqoridagi darslarda ko'rsatib o'tilgan usullar bilan ochishingiz mumkin.
