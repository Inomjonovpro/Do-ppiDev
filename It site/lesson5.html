5-Dars: Funksiyalar (Functions)

Funksiyalar nima?

Funksiyalar (functions) – bu dasturdagi kodlarni takrorlamaslik uchun yaratiladigan bloklar bo‘lib, ular bir marta yoziladi va keyinchalik kerak bo'lganda qayta ishlatiladi. Bu kodni oson boshqarish (manage) va o'qishga yengillashtiradi.

Python tilida funksiyalarni def (define – aniqlash) kalit so'zi bilan e'lon qilamiz.

Funksiya qanday yaratiladi?

Funksiya quyidagi formatda yaratiladi:

def funksiya_nomi():
    # Bu yerda kod yoziladi
    print("Salom, bu funksiya!")

Yuqoridagi misolda biz funksiya_nomi() nomli funksiyani e'lon qildik. Bu funksiya chaqirilganda (called), ekranga "Salom, bu funksiya!" xabarini chiqaradi.

Funksiyani qanday chaqiramiz?

Funksiyani yaratganimizdan so‘ng, uni ishlatish uchun chaqiramiz (call). Funksiyani chaqirish juda oddiy:

funksiya_nomi()  # funksiyani chaqirish

Bu kod funksiyani chaqiradi va uning ichidagi kodni bajaradi.

Funksiya parametrlarini qanday ishlatamiz?

Funksiyalar ma'lumotlarni parametrlar (parameters) yordamida qabul qilishi va ular bilan ishlashi mumkin. Parametrlar funksiyaga qiymatlarni yuborish (pass values) imkonini beradi. 

def salom_ber(ism):  # ism parametri qabul qilinadi
    print(f"Salom, {ism}!")  # ismingiz bilan salom beradi

Bu yerda salom_ber funksiyasi ism parametrini qabul qiladi va o‘sha ism bilan xabar chiqaradi:

salom_ber("Muhammadamin")  # Salom, Muhammadamin!

Funksiyani chaqirganingizda unga kerakli parametrni yuborasiz, va u shu qiymat bilan ishlaydi.

Funksiya qaytariqlari (Return Values)

Ba'zi funksiyalar natijani qaytarishi (return) mumkin. Funksiyaning natijasini qaytarish uchun return kalit so‘zi ishlatiladi. 

def kvadrat_soni(n):
    return n ** 2  # n ning kvadrati qaytariladi

Bu funksiya n qiymatini qabul qilib, uning kvadratini (square) hisoblaydi va natijani qaytaradi.

son = kvadrat_soni(5)  # son = 25
print(son)  # 25 ni chiqaradi

Yuqoridagi kodda kvadrat_soni(5) funksiyasi 5 ning kvadratini qaytarib, natijani son o‘zgaruvchisiga saqlaydi.

Misol (Example)

Keling, funksiyalardan foydalangan holda oddiy misol yozamiz:

def katta_son(a, b):
    if a > b:
        return a  # agar a b dan katta bo'lsa, a qaytariladi
    else:
        return b  # aks holda, b qaytariladi

print(katta_son(10, 20))  # Natija: 20

Bu kod ikki sonni taqqoslaydi va kattasini qaytaradi.
