HTML 2-Dars: HTML Hujjatining Asosiy Tuzilishi

HTML hujjatining tuzilishi

HTML hujjatlari teglar yordamida yoziladi. Har bir HTML hujjati quyidagi asosiy qismlarga ega:

  • <!DOCTYPE html> — HTML5 hujjat ekanligini ko'rsatadi.
  • <html> — HTML hujjatining boshlanishi.
  • <head> — Sahifaning bosh qismi, unda meta ma'lumotlar, sahifa nomi va uslublar ko'rsatiladi.
  • <body> — Sahifaning asosiy tarkibi joylashadigan qism.

Asosiy teglar bilan ishlash

Har bir HTML hujjati <html> va </html> teglar orasida joylashadi. Hujjatning ikki muhim qismi bor: head va body.
Keling, birgalikda har birini ko'rib chiqamiz.

<!DOCTYPE html> va <html> tegi

Hujjatning eng boshida biz <!DOCTYPE html> yozamiz. Bu HTML5 standartiga mosligini bildiradi. Quyida esa <html> tegi bilan hujjatni boshlaymiz. Misol: 

<!DOCTYPE html>
<html lang="uz">
</html>

<head> tegi

<head> qismi sahifaga kiruvchi meta ma'lumotlarni saqlaydi. Unda sahifaning nomi (<title>), belgilar kodirovkasi (<meta charset="UTF-8">) va boshqa meta ma'lumotlar joylashadi. Quyida kichik misol: 

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mening Sahifam</title>
</head>

<body> tegi

Asosiy ma'lumotlarni <body> tegida yozamiz. Bu yerda veb-sahifada ko'rinadigan matn, rasmlar, tugmalar va boshqa elementlar joylashadi. Misol uchun: 

<body>
    <h1>Bu asosiy sarlavha</h1>
    <p>Bu esa oddiy paragraf.</p>
</body>

To'liq hujjat misoli

Quyida to'liq HTML hujjat misolini ko'rishingiz mumkin:

<!DOCTYPE html>
<html lang="uz">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mening Birinchi Sahifam</title>
</head>
<body>
    <h1>Salom, Dunyo!</h1>
    <p>Bu mening birinchi HTML sahifam.</p>
</body>
</html>

Ushbu hujjatni yaratib, brauzerda ochsangiz, sahifada "Salom, Dunyo!" degan matn va oddiy paragrafni ko'rasiz.
