17-Dars: Selenium bilan Avtomatik Testlash

Selenium haqida

Selenium - bu brauzerni avtomatlashtirish uchun ishlatiladigan dasturiy ta'minot vositasi. U avtomatik ravishda veb-sahifalarni ochish, shakllarni to'ldirish, tugmalarni bosish va boshqa ko'plab harakatlarni amalga oshirishi mumkin. Selenium veb-dasturlarni test qilishda juda foydali hisoblanadi.

Kerakli kutubxonalarni o'rnatish

Selenium bilan ishlash uchun kutubxonani quyidagi buyruq orqali o'rnating: 

pip install selenium

Oddiy Selenium dasturi

Quyida Selenium orqali Google'da qidiruv amalga oshirish uchun sodda bir misol keltirilgan: 

from selenium import webdriver
from selenium.webdriver.common.keys import Keys

# Brauzerni ochish
driver = webdriver.Chrome()

# Google'ni ochish
driver.get("https://www.google.com")

# Qidiruv satriga matn kiritish
search_box = driver.find_element("name", "q")
search_box.send_keys("Python dasturlash tili")
search_box.send_keys(Keys.RETURN)

# Natijalar sahifasini chop etish
print(driver.title)

# Brauzerni yopish
driver.quit()

Webdrayverni o'rnatish

Selenium ishlashi uchun sizga brauzer uchun WebDriver kerak bo'ladi. Masalan, agar siz Google Chrome ishlatayotgan bo'lsangiz, ChromeDriver o'rnatishingiz kerak bo'ladi. ChromeDriver'ni bu yerda yuklab olishingiz mumkin.

Elementlarni topish

Selenium'da elementlarni topish uchun bir nechta usullar mavjud:

  • find_element_by_name() - Elementni nomi bo'yicha topish.
  • find_element_by_id() - Elementni ID'si bo'yicha topish.
  • find_element_by_xpath() - XPath yordamida elementni topish.

Keyingi qadamlar

Bu darsda Selenium orqali oddiy avtomatlashtirilgan testni yaratishni o'rgandingiz. Keyingi darslarda Selenium'ni yanada murakkabroq testlash jarayonlari uchun ishlatishni o'rganamiz.
