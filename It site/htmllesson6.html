JavaScript Bilan Tanishing

JavaScript Nima?

JavaScript (JS) — veb-sahifalar uchun ishlatiladigan dasturlash tilidir. U dinamik, kuchli va ob'ektga yo'naltirilgan dasturlashni qo'llab-quvvatlaydi.

JavaScript Misoli

JavaScript yordamida veb-sahifalarda interaktivlik qo'shishingiz mumkin. Quyida oddiy JavaScript kodi keltirilgan:

console.log("Salom, Dunyo!");

JavaScriptdan Foydalanish

JavaScript yordamida sahifalarga turli funksiyalar qo'shishingiz mumkin, masalan, tugmalarga bosish, formalarni tasdiqlash va boshqa ko'plab vazifalar.

Misol uchun:

function salomBer() {
    alert("Salom, bu JavaScript misoli!");
}
