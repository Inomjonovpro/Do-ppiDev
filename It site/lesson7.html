7-Dars: While Tsikli (While Loop)

While Tsikli nima?

While tsikli - bu kodni biror shart (condition) to‘g‘ri bo‘lganda (haqiqat bo‘lganda) takrorlab ishlatadi. While tsikli to‘xtashi uchun shart noto‘g‘ri (false) bo‘lishi kerak.

While Tsikli qanday ishlaydi?

While tsikli biror shartni tekshiradi va shart to‘g‘ri bo‘lsa, kodni qayta-qayta bajaradi. Shart noto‘g‘ri bo‘lsa, tsikl to‘xtaydi. 

x = 1
while x <= 5:  # shart: x 5 dan kichik yoki teng bo'lsa
    print(x)   # x ni ekranga chiqaramiz
    x += 1     # har safar x ga 1 qo'shamiz

Bu misolda while tsikli 5 marta ishlaydi va x ning qiymatini ekranga chiqaradi. Har safar x 1 ga oshiriladi.

Cheksiz While Tsiklidan saqlaning

Agar while tsiklidagi shart hech qachon noto‘g‘ri bo‘lmasa, tsikl hech qachon to‘xtamaydi. Bu holatga **cheksiz tsikl (infinite loop)** deyiladi va kompyuter ishidagi xatolarga sabab bo‘lishi mumkin. 

x = 1
while x > 0:  # bu shart doimo to'g'ri bo'ladi, tsikl hech qachon to'xtamaydi
    print(x)

Bu misolda tsikl hech qachon to‘xtamaydi, chunki shart doimo haqiqat bo‘lib turadi.

Break yordamida while tsiklni to‘xtatish

Agar while tsiklni ma'lum bir joyda to‘xtatmoqchi bo‘lsangiz, break kalit so‘zidan foydalanishingiz mumkin. Bu kalit so‘z tsiklni to‘xtatadi va undan tashqaridagi kodni bajaradi. 

x = 1
while x <= 10:
    if x == 6:  # x 6 ga teng bo'lganda tsikl to'xtaydi
        break
    print(x)
    x += 1

Bu yerda tsikl 6 ga yetganda break so‘zi tufayli to‘xtatiladi.

Continue yordamida tsiklning ma'lum bir qismini o'tkazib yuborish

continue kalit so‘zi while tsikl ichidagi biror qismni o‘tkazib yuboradi va tsiklning keyingi iteratsiyasiga (takrorlanishiga) o‘tadi. 

x = 0
while x < 5:
    x += 1
    if x == 3:  # x 3 ga teng bo'lganda davom etadi, 3 chiqarilmaydi
        continue
    print(x)

Bu yerda x 3 bo‘lganda continue so‘zi tufayli o‘tkazib yuboriladi, lekin qolgan barcha sonlar ekranga chiqariladi.

Misol (Example)

While tsikldan foydalanib oddiy misol yozamiz:

son = 1
while son <= 10:
    print(f"{son} * 3 = {son * 3}")
    son += 1

Bu yerda biz 1 dan 10 gacha sonlarni olamiz va ularni 3 ga ko‘paytirib natijani chiqaramiz.

Xulosa

While tsikli - bu kodni takrorlashda muhim vositadir. Uning yordamida shart bajarilgunga qadar bir xil kodni takrorlash mumkin. Keyingi darsimizda OOP (Ob'ektga yo'naltirilgan dasturlash) haqida o‘rganamiz.
