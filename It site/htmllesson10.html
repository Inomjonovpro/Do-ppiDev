O'z Sahifangizni Yaratish

Sahifa Nima?

Veb-sahifa — bu Internetda joylashgan hujjat bo'lib, u foydalanuvchilar uchun ma'lumotlarni taqdim etadi. HTML, CSS va JavaScript yordamida sahifalar yaratiladi.

HTML Kod Misoli

Quyida oddiy HTML sahifasini yaratish uchun kerakli kod keltirilgan:

<!DOCTYPE html>
<html lang="uz">
<head>
    <meta charset="UTF-8">
    <title>Sahifangiz</title>
</head>
<body>
    <h1>Salom, dunyo!</h1>
</body>
</html>

CSS Qo'llash

CSS yordamida sahifalaringizni go'zal ko'rinishga keltirishingiz mumkin. Masalan, quyidagi CSS kodini qo'shish orqali sahifangizning fon rangini o'zgartirishingiz mumkin: 

body {
    background-color: #e0f7fa;
}
