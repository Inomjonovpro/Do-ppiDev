14-Dars: Sodda Dasturlar Yaratish

Python orqali sodda dasturlar yaratish

Bu darsda siz oddiy Python dasturlarini yaratishni o'rganasiz. O'zingiz ishlatishingiz mumkin bo'lgan oddiy dasturlar misollari bilan tanishamiz, masalan, kalkulyator, raqam taxmin qilish o'yini va boshqalar.

Kerakli kutubxonalarni o'rnatish

Dasturlarni yaratishda foydali kutubxonalar kerak bo'lishi mumkin. Agar dasturning ishlashi uchun kutubxonalar kerak bo'lsa, ularni quyidagi tarzda o'rnatishingiz mumkin:

Kutubxonalarni o'rnatish: Terminal yoki buyruqlar satriga quyidagi buyruqlarni kiriting:

pip install requests
pip install random

requests moduli API bilan ishlash uchun, random moduli esa tasodifiy raqamlar yaratish uchun ishlatiladi.

Oddiy Kalkulyator Dasturi

Quyida oddiy kalkulyator dasturining kodi keltirilgan. Bu dastur foydalanuvchidan ikkita son va amalni so'raydi va natijani hisoblaydi. 

def kalkulyator():
    birinchi_son = float(input("Birinchi sonni kiriting: "))
    ikkinchi_son = float(input("Ikkinchi sonni kiriting: "))
    amal = input("Amalni tanlang (+, -, *, /): ")

    if amal == "+":
        print(f"Natija: {birinchi_son + ikkinchi_son}")
    elif amal == "-":
        print(f"Natija: {birinchi_son - ikkinchi_son}")
    elif amal == "*":
        print(f"Natija: {birinchi_son * ikkinchi_son}")
    elif amal == "/":
        if ikkinchi_son != 0:
            print(f"Natija: {birinchi_son / ikkinchi_son}")
        else:
            print("Bo'lish amali nolga bo'lish mumkin emas!")
    else:
        print("Noto'g'ri amal tanlandi!")

if __name__ == "__main__":
    kalkulyator()

Random Raqam Topish O'yini

Ushbu dasturda kompyuter tasodifiy raqam tanlaydi va foydalanuvchi o'sha raqamni topishi kerak bo'ladi. 

import random

def raqam_top():
    max_son = 100
    tasodifiy_son = random.randint(1, max_son)
    taxminlar = 0
    taxmin = -1

    while taxmin != tasodifiy_son:
        taxmin = int(input(f"1 dan {max_son} gacha son kiriting: "))
        taxminlar += 1
        if taxmin < tasodifiy_son:
            print("Ko'proq bo'lishi kerak.")
        elif taxmin > tasodifiy_son:
            print("Kamroq bo'lishi kerak.")
    
    print(f"Tabriklaymiz! Siz {taxminlar} taxmin bilan to'g'ri topdingiz!")

if __name__ == "__main__":
    raqam_top()

Vaqt va Sana Ko'rsatuvchi Dastur

Quyidagi dastur hozirgi vaqt va sanani ko'rsatadi. Bu datetime kutubxonasi yordamida amalga oshiriladi. 

import datetime

def vaqt_va_sana():
    hozirgi_vaqt = datetime.datetime.now()
    print("Hozirgi vaqt va sana:", hozirgi_vaqt)

if __name__ == "__main__":
    vaqt_va_sana()

API bilan Ma'lumot Olish Dasturi

Ushbu dasturni yaratishda requests kutubxonasidan foydalanib, internetdan ma'lumotlar olamiz. Masalan, ob-havo ma'lumotlarini olish. 

import requests

def ob_havo():
    shahar = input("Shahar nomini kiriting: ")
    api_url = f"http://api.openweathermap.org/data/2.5/weather?q={shahar}&appid=SIZNING_API_KALITingiz"
    
    javob = requests.get(api_url)
    if javob.status_code == 200:
        data = javob.json()
        print(f"Shahar: {data['name']}, Harorat: {data['main']['temp']} K")
    else:
        print("Ma'lumot topilmadi!")

if __name__ == "__main__":
    ob_havo()
