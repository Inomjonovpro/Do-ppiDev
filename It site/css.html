CSS Dasturlash Kursi

CSS (Cascading Style Sheets) - bu web sahifalarning ko'rinishini boshqarish uchun ishlatiladigan dasturlash tili. CSS, HTML bilan birga, web dasturlashning asosiy komponentlaridan biridir. CSS 1996-yilda W3C tomonidan yaratilgan.

CSS yordamida siz web sahifalarning ranglarini, shriftlarini, maketini va boshqa ko'plab dizayn elementlarini belgilashingiz mumkin. CSS, elementlar o'rtasidagi masofalarni, o'lchamlarni va shakllarni boshqarish imkonini beradi.

CSS ning yangi versiyalari (CSS3) ko'plab yangi imkoniyatlarni, jumladan, animatsiyalar, gradientlar va media queries qo'shishni ta'minlaydi.

Darslar yaqin orada qo'shiladi!
