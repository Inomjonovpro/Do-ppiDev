CSS Bilan Tanishing

CSS Nima?

CSS (Cascading Style Sheets) veb-sahifalarning ko'rinishini boshqarish uchun ishlatiladigan uslub varaqasidir. CSS yordamida siz matnning rangi, fon rangi, joylashuvi va boshqa ko'plab uslublarni belgilashingiz mumkin.

CSS Misoli

CSS yordamida veb-sahifalarning ko'rinishini o'zgartirish mumkin. Quyida HTML va CSS orqali yaratilgan misol rasmga qarang:

Ushbu rasm HTML yordamida yaratilgan oddiy sahifani ko'rsatadi.

Ushbu rasm HTML va CSS yordamida yanada chiroyli ko'rinishga ega sahifani ko'rsatadi.

CSSdan Foydalanish

CSS yordamida siz veb-sahifangizga ko'plab uslublar qo'shishingiz mumkin, masalan, fon rangi, matn rangi, shrifti, margin va padding kabi xususiyatlar.

Misol uchun:

body {
    background-color: #f0f0f0;
    color: #333;
    font-family: 'Arial', sans-serif;
}
