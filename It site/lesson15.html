Oddiy Ma'lumotlar Tahlili

Quyida Pandas yordamida ma'lumotlarni tahlil qilishning oddiy namunasi keltirilgan. Pandas yordamida CSV formatidagi fayllarni o'qib, ularni qayta ishlash mumkin. 

import pandas as pd

# CSV faylini o'qish
data = pd.read_csv('example.csv')

# Birinchi 5 qatorni ko'rsatish
print(data.head())

Oddiy Regression Modeli

Quyidagi misolda scikit-learn kutubxonasi yordamida oddiy regression modeli yaratiladi. 

from sklearn.linear_model import LinearRegression
import numpy as np

# Ma'lumotlarni yaratish
x = np.array([1, 2, 3, 4, 5]).reshape(-1, 1)
y = np.array([2, 4, 6, 8, 10])

# Modelni yaratish
model = LinearRegression()
model.fit(x, y)

# Bashorat qilish
predictions = model.predict(x)
print("Bashoratlar:", predictions)

Ma'lumotlarni Vizualizatsiya Qilish

Ma'lumotlarni vizualizatsiya qilish uchun Matplotlib kutubxonasidan foydalanamiz. 

import matplotlib.pyplot as plt

# Ma'lumotlarni chizish
plt.plot(x, y, label='Asl ma\'lumotlar')
plt.plot(x, predictions, label='Bashorat qilingan')
plt.xlabel('X qiymatlari')
plt.ylabel('Y qiymatlari')
plt.title('Regression modeli')
plt.legend()
plt.show()
