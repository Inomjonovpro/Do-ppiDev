21-Dars: Containerization (Docker)

1. Containerization nima?

Containerization - bu dasturiy ta'minotni va uning barcha bog'liqliklarini bitta konteynerga joylashtirish jarayoni. Bu dastur muhitini boshqa tizimlarda osonlikcha ko'chirib, ishlatishga imkon beradi. Docker - bu containerization uchun eng mashhur vositalardan biridir.

2. Docker o'rnatish

Docker'ni o'rnatish uchun quyidagi buyruqlarni terminalda ishlating (operatsion tizimingizga qarab farq qilishi mumkin): 

# Ubuntu/Linux tizimida Docker o'rnatish
sudo apt update
sudo apt install docker.io

# Windows yoki Mac OS'da Docker Desktop'ni o'rnatish uchun rasmiy saytga boring:
# https://www.docker.com/products/docker-desktop

3. Dockerfile yaratish

Docker'da dasturiy ta'minotni joylashtirish uchun Dockerfile - bu dasturiy ta'minotni qanday paketlashni ko'rsatuvchi fayl. Keling, oddiy Python dasturini Docker'ga joylashtirish jarayonini ko'rib chiqamiz.

Dockerfile misoli:

# Dockerfile
# Python bazasi asosida ish boshlaymiz
FROM python:3.9

# Ilova fayllarini konteynerga nusxalash
WORKDIR /app
COPY . /app

# Bog'liqliklarni o'rnatish
RUN pip install -r requirements.txt

# Ilovani ishga tushirish
CMD ["python", "main.py"]

Ilovani Docker'da ishga tushirish:

# Dockerfile yaratgandan keyin, konteyner yaratamiz
docker build -t mening_ilovam .

# Konteynerni ishga tushiramiz
docker run -d -p 5000:5000 mening_ilovam

4. Konteynerlarni boshqarish

Docker konteynerlarining ish holatini tekshirish uchun quyidagi buyruqlardan foydalanishingiz mumkin:

  • docker ps - ishlayotgan konteynerlarni ko'rsatadi.
  • docker stop [container_id] - konteynerni to'xtatadi.
  • docker rm [container_id] - konteynerni o'chiradi.

Xulosa

Ushbu darsda containerization tushunchasi va Docker vositasi bilan tanishdik. Docker yordamida dasturlarni izolyatsiya qilish va ularni turli muhitlarda barqaror ishlashini ta'minlash mumkin. Bu dasturlarni tarqatish va ishlab chiqishni tezlashtiradi.
