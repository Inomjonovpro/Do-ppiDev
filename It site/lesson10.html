10-Dars: API'lar va Modullar bilan ishlash

API nima?

API (Application Programming Interface) dasturiy ta'minotlar orasida muloqot qilish uchun mo'ljallangan interfeysdir. API'lar boshqa dasturiy ta'minotlarning funksiyalarini chaqirishga imkon beradi.

API'lar qanday ishlaydi?

API'lar odatda HTTP protokoli orqali ishlaydi va ma'lumotlar JSON yoki XML formatlarida uzatiladi.

Python'da API'lar bilan ishlash

Python'da API'lar bilan ishlash uchun requests kutubxonasi keng qo'llaniladi. Quyidagi misolda API'dan ma'lumot olish jarayoni ko'rsatilgan: 

import requests

# API URL'si
url = 'https://api.exapmle.com/data'

# API'dan ma'lumot olish
response = requests.get(url)

# Javobni tekshirish
if response.status_code == 200:
    data = response.json()
    print(data)
else:
    print("Xato:", response.status_code)

Foydalanish misoli

Misol uchun, o'zingizning weather API'ingizdan foydalanib, ob-havoni olish: 

import requests

# OpenWeatherMap API URL'si
api_key = 'SIZNING_API_KALITINGIZ'
city = 'Toshkent'
url = f'http://api.openweathermap.org/data/2.5/weather?q={city}&appid={api_key}&units=metric'

response = requests.get(url)

if response.status_code == 200:
    data = response.json()
    print(f"{city} ob-havosi: {data['main']['temp']}°C")
else:
    print("Xato:", response.status_code)

Modullar haqida

Python'da modullar dasturlarni tashkil etish va kodni qayta ishlatish imkonini beradi. Har bir modul o'z ichida funksiyalar, klasslar va o'zgaruvchilarni saqlaydi.

O'z modulini yaratish uchun, avvalo, yangi fayl yaratish va unga .py kengaytmasini berish kerak. Masalan, my_module.py.

Modulni o'rnatish

Ommabop modullarni o'rnatish uchun pip paket menejeridan foydalanamiz. Quyidagi buyruq yordamida requests modulini o'rnatish mumkin: 

pip install requests

O'rnatilgan modulni dasturda quyidagicha import qilamiz: 

import requests

Keyingi darslar

Keyingi darslarda asinxron dasturlash (asyncio) haqida ma'lumot olasiz.
